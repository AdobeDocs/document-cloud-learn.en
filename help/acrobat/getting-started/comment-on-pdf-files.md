Learn how to add comments on a shared PDF using just a web browser. Whether you're collaborating with a team or proofreading a document, Acrobat's commenting tools make it simple to provide clear and actionable feedback. This video tutorial uses the [New Acrobat experience](new-experience.md).

To learn how to share a PDF, check out the [Collaborate in real time](collaborate.md) tutorial.
