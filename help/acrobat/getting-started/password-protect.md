# Protect a PDF file with a password

Learn how to [add a password to your PDF](https://www.adobe.com/acrobat/online/password-protect-pdf.html){target="_blank"} to protect others from opening or [editing the PDF file](https://www.adobe.com/acrobat/online/pdf-editor.html){target="_blank"}. Protecting your PDF files prevents unauthorized access, protects sensitive information, maintains document integrity, and helps comply with certain industry regulations.

>[!SLIDE](protect-pdfs)

>[!TIP]
>
>To password protect multiple PDF files, try using the [Action Wizard](../advanced-tasks/action.md).
