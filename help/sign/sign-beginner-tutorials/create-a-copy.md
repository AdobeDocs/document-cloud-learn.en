# When to use Create a copy

Already sent an agreement — but it expired, was canceled, or you need to send it again? Create a copy lets you duplicate any existing agreement, adjust what needs changing, and resend it in seconds. No rebuilding from scratch. Create a copy is available when:

* You are the original sender of the agreement
* The agreement was sent from the Modern Request Signature Experience
* The agreement does not include a witness recipient role
* The agreement has not been deleted due to document retention or GDPR deletion
* The agreement does not require access to a library template the sender no longer has access to

>[!NOTE]
>
>Create a copy is available to enterprise-level plans.
