# Create mobile-friendly view

Learn how to create a mobile-friendly view of the document with "Mobile Focus" mode. Senders can now generate a mobile-friendly form seamlessly, without any developer support.

With Mobile Focus, senders benefit from:

* Increased completion rate: Higher form completion rates leading to improved outcomes
* Reduce time and cost: Avoid the effort required to convert documents into mobile-friendly forms
* No technical expertise needed: Generate dynamic, mobile-friendly forms with minimal effort
