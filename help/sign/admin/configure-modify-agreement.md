# Configure and govern Modify Agreement

Learn how to configure and govern **Modify Agreement** in Acrobat Sign. In this tutorial, administrators set account‑ and group‑level controls, understand eligibility rules, and learn how agreement integrity is preserved when changes are made after sending. See how sender actions are tracked and audited to support compliance and transparency.

What you'll learn:

* Which admin settings enable or restrict agreement modification
* When agreements can and cannot be modified
* How revision mode protects agreement integrity
* What changes are logged in the audit report
